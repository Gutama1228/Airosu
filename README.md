# Airosu

AI-powered assistant for Roblox Studio developers.

## Features
- Public website with AI playground
- Admin dashboard (mock)
- Built with React, TypeScript, Vite, Tailwind CSS
- Deployed on Vercel

## Setup
1. `npm install`
2. `npm run dev`
3. Visit `http://localhost:5173`
